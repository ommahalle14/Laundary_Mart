# Laundry Mart 🧺

Laundry Mart is a modern and user-friendly laundry service website designed to provide customers with a seamless laundry experience. The platform showcases various laundry services, pricing plans, and business information in a clean and responsive web interface.

## Features

- Professional and responsive website design
- Service and pricing information
- Laundry, Dry Cleaning, and Ironing services
- Doorstep Pickup & Delivery details
- Customer-friendly user interface
- Mobile and desktop compatible layout

## Services Offered

- Regular Wash & Fold
- Wash & Iron
- Premium Laundry Care
- Dry Cleaning
- Ironing Services
- Bedsheet & Blanket Cleaning
- Curtain Cleaning
- Express Laundry Service
- Doorstep Pickup & Delivery

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Objective

The objective of this project is to create a simple yet attractive website for a laundry business that helps customers explore services, view pricing, and learn more about Laundry Mart. This project demonstrates front-end web development skills and responsive design principles.
